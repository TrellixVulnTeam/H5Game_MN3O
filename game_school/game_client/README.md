放学别走游戏框架简要说明

目录结构
===============
resource：主要存放资源
	asstes：存放一些原始资源 例如：.png/.zip/.jpg等等
	config：存放策划配置的表格数据
	effect：存放特效
	eui_assets：存放eui库用到的资源
	eui_skins：存放使用eui资源包装的皮肤文件
	font：存放策划打包的字体
	proto：存放前后端通信的proto文件
	skin：存放项目用到的皮肤文件，例如：登录，大厅等等

scripts：封装的一些命令文件

src：存放代码
	core：核心代码（MVC框架，资源处理，音效管理，龙骨，场景管理，通信等）
	
End
===============
